\mybanner{Álgebra computacional. Examen de recuperación}{Universidad de El Salvador, 20/06/2019}

Conteste \textbf{brevemente} a las preguntas de abajo. Cada respuesta correcta
vale 1 punto.

\begin{enumerate}
\item Defina qué es una \term{base de Gröbner} de un ideal
  $I \subset k [x_1,\ldots,x_n]$. Formule el \term{criterio de Buchberger} y
  describa el \term{algoritmo de Buchberger}.

\item Defina qué es una \term{base de Gröbner reducida} y describa cómo
  calcularla.

\item Describa cómo a partir de una base de Gröbner de $I$ encontrar una base de
  $k [x_1,\ldots,x_n]/I$ como un $k$-espacio vectorial.

\item Explique cómo para un ideal $I \subseteq k [x_1,\ldots,x_n]$ se puede
  calcular el ideal $I_\ell \dfn I \cap k [x_{\ell+1},\ldots,x_n]$.

\item Explique cómo para dos ideales $I,J \subseteq k [x_1,\ldots,x_n]$ se puede
  calcular la intersección $I \cap J$.

\item Defina la \term{topología de Zariski}; \term{subconjuntos} y
  \term{componentes irreducibles} de un conjunto algebraico
  $X \subseteq \AA^n (k)$.

\item Formule los dos \term{teoremas de unicidad} para las
  \term{descomposiciones primarias}.

\item Defina la \term{dimensión de Krull} de un anillo conmutativo y explique
  cómo calcularla para $k$-álgebras finitamente generadas.

\item Defina la \term{serie de Hilbert} $H_I (t)$ y explique cómo calcularla
  usando una base de Gröbner para $I$.

\item Defina qué es una \term{normalización de Noether} y explique cómo
  encontrarla.
\end{enumerate}
